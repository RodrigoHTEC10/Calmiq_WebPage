## Calmiq
Calmiq se plantea como la experiencia de acompañamiento y guia para el cuidado y la autoregulación del estrés en base a un software interactivo cuya mascota Mizu reaccione en base al estado del usuario y un prototipo de aparicencia de guante el cual a partir de sensados biológicos pueda determinar el nivel de estrés de una persona.
